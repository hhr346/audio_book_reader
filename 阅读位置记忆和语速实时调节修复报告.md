## ✅ 修复方案

### 修复 1: 阅读位置记忆

**问题分析**:
- `_loadCurrentChapter()` 方法中只处理了书签和强制跳转
- 没有从 `widget.book.currentPageIndex` 恢复上次位置
- 默认回到第一页（`restorePageIndex = 0`）

**修改文件**: `lib/screens/reader_screen.dart`

**修改内容**:
```dart
// 恢复页面位置
int restorePageIndex = 0;

if (widget.initialPageIndex != null && widget.initialPageIndex! < pages.length) {
  // 从书签打开
  restorePageIndex = widget.initialPageIndex!;
  print('📑 从书签恢复：第${_currentChapterIndex + 1}章 第${restorePageIndex + 1}页');
} else if (forceFirstPage) {
  // 强制跳转到第一页（章节切换时）
  restorePageIndex = 0;
  print('📖 跳转到新章节：第${_currentChapterIndex + 1}章 第 1 页');
} else if (goToLastPage) {
  // 跳转到最后一页（从下一章返回上一章时）
  restorePageIndex = pages.length - 1;
  print('📖 跳转到最后一页：第${_currentChapterIndex + 1}章 第${restorePageIndex + 1}页');
} else {
  // ✨ 新增：恢复上次阅读位置（从 book 模型中读取）
  restorePageIndex = widget.book.currentPageIndex;
  // 确保页码不超出当前章节的实际页数
  if (restorePageIndex >= pages.length) {
    restorePageIndex = pages.length - 1;
    print('⚠️ 上次页码超出范围，调整为最后一页');
  }
  print('📖 恢复上次阅读位置：第${_currentChapterIndex + 1}章 第${restorePageIndex + 1}页');
}
```

**恢复优先级**:
1. **书签位置** - 从书签打开时
2. **上次位置** - 正常打开书籍时（新增）
3. **第一章** - 第一次阅读时

**数据流**:
```
用户退出 → 保存进度 → book.currentPageIndex
           ↓
用户打开 → 读取 book.currentPageIndex → 恢复位置
```

**效果**:
- ✅ 退出后重新打开，自动恢复到上次阅读位置
- ✅ 页码超出范围时自动调整到最后一页
- ✅ 书签打开优先于上次位置

### 修复 2: 语速实时调节

**问题分析**:
- 原代码：`onChanged` 只更新本地变量，点击"确定"才调用 `setRate`
- 用户需要拖动 → 确定 → 才能听到语速变化
- 无法即时预览不同语速的效果

**修改文件**: `lib/screens/settings_screen.dart`

**修改内容**:

#### 修改前
```dart
void _showSpeechRateDialog() {
  final ttsService = TtsService();
  double rate = ttsService.speechRate;
  
  showDialog(
    context: context,
    builder: (context) => AlertDialog(
      title: const Text('语速设置'),
      content: StatefulBuilder(
        builder: (context, setDialogState) => Column(
          children: [
            Slider(
              value: rate,
              onChanged: (value) {
                setDialogState(() => rate = value);
                // ❌ 只更新本地变量
              },
            ),
            Text('当前语速：${(rate * 100).round()}%'),
          ],
        ),
      ),
      actions: [
        TextButton(onPressed: () => Navigator.pop(context), ...),
        ElevatedButton(
          onPressed: () async {
            await ttsService.setRate(rate); // ❌ 点击确定才设置
            Navigator.pop(context);
          },
          ...
        ),
      ],
    ),
  );
}
```

#### 修改后
```dart
void _showSpeechRateDialog() {
  final ttsService = TtsService();
  
  showDialog(
    context: context,
    builder: (context) => StatefulBuilder(
      builder: (context, setDialogState) => AlertDialog(
        title: const Text('语速设置'),
        content: Column(
          children: [
            Slider(
              value: rate,
              onChanged: (value) async {
                setDialogState(() => rate = value);
                // ✅ 实时调整语速
                await ttsService.setRate(value);
              },
            ),
            Text('当前语速：${(rate * 100).round()}%', 
              style: const TextStyle(fontSize: 16, fontWeight: FontWeight.bold)),
          ],
        ),
        actions: [
          TextButton(onPressed: () => Navigator.pop(context), ...),
          ElevatedButton(
            onPressed: () => Navigator.pop(context), // ✅ 直接关闭
            ...
          ),
        ],
      ),
    ),
  );
}
```

**关键改动**:
1. **StatefulBuilder 外层包裹** - 使整个对话框可以状态更新
2. **onChanged 中调用 setRate** - 拖动时实时调整
3. **确定按钮简化** - 只需关闭对话框

**效果**:
- ✅ 拖动进度条时语速实时变化
- ✅ 可以即时预览不同语速效果
- ✅ 调整完成后直接点"确定"关闭
