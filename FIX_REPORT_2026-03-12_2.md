## 🔴 修复的问题

### 1. 夜间模式切换不生效

**问题描述**:
- 在阅读器内点击右上角设置 → 切换夜间模式 → 当前页面颜色不改变
- 返回书架后主题才改变

**根本原因**:
- 阅读器页面的 `_isDarkMode` 状态只在 `initState` 时从 SharedPreferences 读取
- 设置页面的 `onThemeChanged` 回调只更新了本地状态，没有同步到 SharedPreferences
- 阅读器页面没有监听主题变化

**修复方案**:
1. 在 `_openSettings()` 方法中，`onThemeChanged` 回调立即保存到 SharedPreferences
2. 设置页面返回后调用 `_loadSettings()` 重新加载主题设置
3. 确保主题切换即时生效

**修改文件**:
- `lib/screens/reader_screen.dart` - `_openSettings()` 方法

**代码变更**:
```dart
void _openSettings() async {
  await Navigator.push(
    context,
    MaterialPageRoute(
      builder: (context) => SettingsScreen(
        onThemeChanged: (isDark) async {
          // 立即更新本地状态
          setState(() => _isDarkMode = isDark);
          // 保存到 SharedPreferences
          final prefs = await SharedPreferences.getInstance();
          await prefs.setBool('is_dark_mode', isDark);
        },
        ...
      ),
    ),
  );
  // 返回后重新加载设置（确保同步）
  await _loadSettings();
}
```

### 2. 章节跳转继承上一页页码

**问题描述**:
- 从第 3 章（在第 5 页）跳转到第 4 章 → 第 4 章显示第 5 页（应该是第 1 页）
- 章节列表跳转也有同样问题

**根本原因**:
- `_loadCurrentChapter()` 方法总是尝试恢复 `widget.book.currentPageIndex`
- 没有区分"继续上次阅读"和"跳转到新章节"两种场景
- 新章节应该总是从第 1 页开始

**修复方案**:
1. `_loadCurrentChapter()` 添加 `forceFirstPage` 参数
2. 当 `forceFirstPage = true` 时，忽略上次阅读位置，强制跳转到第 1 页
3. `_nextChapter()` 和章节列表跳转时传入 `forceFirstPage: true`

**修改文件**:
- `lib/screens/reader_screen.dart` - `_loadCurrentChapter()`, `_nextChapter()`, `_showChapterList()`

**代码变更**:
```dart
// 添加 forceFirstPage 参数
Future<void> _loadCurrentChapter({bool forceFirstPage = false}) async {
  ...
  if (widget.initialPageIndex != null && widget.initialPageIndex! < pages.length) {
    // 从书签打开
    restorePageIndex = widget.initialPageIndex!;
  } else if (forceFirstPage) {
    // 强制跳转到第一页（章节切换时）
    restorePageIndex = 0;
  }
  ...
}

// 下一章时强制跳转到第一页
Future<void> _nextChapter() async {
  if (_currentChapterIndex < widget.book.totalChapters - 1) {
    setState(() {
      _currentChapterIndex++;
      _currentPageIndex = 0;
    });
    await _loadCurrentChapter(forceFirstPage: true); // 强制第一页
  }
}

// 章节列表跳转也强制第一页
onTap: () {

    _currentChapterIndex = index;
    _currentPageIndex = 0;
  });
  _loadCurrentChapter(forceFirstPage: true);

}
```

- **改进**: 使用参数明确表达意图（`forceFirstPage`）
